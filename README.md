# Student Grade Tracker

## Overview

Student Grade Tracker is a Java console application developed as part of the CodeAlpha Internship Program. The application allows users to enter student details and marks, calculate percentages, assign grades, determine pass/fail status, and generate a complete class performance report.

## Features

* Add multiple students
* Enter marks for Physics, Chemistry, and Biology
* Calculate percentage automatically
* Assign grades based on percentage
* Display pass/fail status
* Calculate class average percentage
* Identify the top performer
* Generate a formatted student report

## Technologies Used

* Java
* ArrayList
* Scanner Class
* Object-Oriented Programming Concepts

## Grading System

| Percentage | Grade |
| ---------- | ----- |
| 90+        | A+    |
| 80-89      | A     |
| 70-79      | B     |
| 60-69      | C     |
| 40-59      | D     |
| Below 40   | F     |

## How to Run

1. Clone the repository:
   git clone https://github.com/your-username/CodeAlpha_StudentGradeTracker.git

2. Compile the program:
   javac StudentGradeTracker.java

3. Run the program:
   java StudentGradeTracker
